# gsorder

`gsorder` is a command line tool that used for reorders the fields of structure in Go(lang) to optimize(minimize) the amount of padding.

## Installation

```bash
$go get -u github.com/barryz/gsorder
```

## Usage

```bash
$gsorder -pkg $your_go_pkg
```
